DEPLOYMENT STEPS:
════════════════════════════════════════════════════════════════

1. PUSH TO GITHUB:
   cd /tmp/kenniswiser-consulting
   git init
   git add .
   git commit -m "Initial: Kennis Wiser Consulting website"
   git remote add origin https://github.com/[username]/kenniswiser-consulting.git
   git push -u origin main

2. DEPLOY ON VERCEL:
   a) Visit https://vercel.com
   b) Click "New Project"
   c) Select GitHub repo: kenniswiser-consulting
   d) Click "Deploy" (uses vercel.json config)

3. CONFIGURE DOMAIN:
   a) In Vercel: Settings → Domains → Add kenniswiserconsulting.com
   b) In Cloudflare: Update DNS to Vercel nameservers
      - ns1.vercel-dns.com
      - ns2.vercel-dns.com

4. WAIT FOR PROPAGATION:
   - DNS updates typically take 24-48 hours
   - SSL certificate auto-issued by Vercel
